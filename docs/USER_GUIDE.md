## 🚀 Быстрый старт

### Установка

1. **Скачайте приложение**:
   - Windows: `CalendarApp.exe`
   - macOS: `CalendarApp.app`
   - Linux: `calendar_app`

2. **Запустите приложение**:
   - Дважды кликните по иконке
   - При первом запуске автоматически создастся папка с базой данных

3. **Создайте первое событие**:
   - Нажмите кнопку **"Событие"** в правом верхнем углу
   - Заполните название, дату, время
   - Выберите категорию
   - Нажмите **"Сохранить"**

Готово! Ваше событие появится в календаре.

## 📝 Управление событиями

### Создание события

**Способ 1: Кнопка "Событие"**
1. Нажмите **"Событие"** в правом верхнем углу
2. Заполните форму:
   - **Название** (обязательно)
   - **Дата** (календарь)
   - **Время начала** (HH:MM)
   - **Время окончания** (HH:MM)
   - **Категория** (выберите из списка)
   - **Описание** (необязательно)
3. Нажмите **"Сохранить"**

**Способ 2: Двойной клик в календаре**
- В режиме **Месяц**: дважды кликните по пустой ячейке
- В режиме **День/Неделя**: дважды кликните по временной шкале
- Откроется та же форма создания

**Способ 3: Быстрое создание в режиме "Список"**
- Нажмите на дату в списке (пока не реализовано)

### Редактирование события

1. **Откройте событие**:
   - В режиме **Месяц**: кликните по событию в ячейке
   - В режиме **День/Неделя**: кликните по блоку события
   - В режиме **Список**: дважды кликните по строке

2. **Измените нужные поля**

3. **Нажмите "Сохранить"**

### Удаление события

1. Откройте событие (см. выше)
2. Нажмите красную кнопку **"Удалить"**
3. Подтвердите удаление в диалоговом окне

## 👁️ Режимы просмотра

### 1. Список (ListView)

![Список]

**Особенности:**
- Все события, сгруппированные по датам
- Заголовки: "Сегодня", "Завтра", "Понедельник, 15 января 2024"
- Прошедшие события **зачёркнуты** и имеют полупрозрачный цвет
- При наведении строка подсвечивается

**Действия:**
- **Двойной клик** по событию → редактирование
- **Клик** по заголовку даты → (прокрутка к сегодня)

**Совет**: Используйте для быстрого просмотра всех событий.

### 2. День (DayView)

![День]

**Особенности:**
- Временная шкала с 00:00 до 23:00
- Каждый час разделён на две половины (пунктирная линия)
- **Красная линия** показывает текущее время (только для сегодня)
- События отображаются цветными блоками

**Визуальные эффекты:**
- Цвет блока соответствует категории
- Левая полоска - цвет категории
- Пересекающиеся события автоматически группируются

**Действия:**
- **Двойной клик** по пустому месту → новое событие
- **Клик** по событию → редактирование
- **Прокрутка** мыши для перемещения по времени

**Совет**: Используйте для детального планирования дня.

### 3. Неделя (WeekView)

![Неделя]

**Особенности:**
- Показывает **Понедельник – Пятницу** (рабочая неделя)
- Временная шкала как в режиме "День"
- **Красная линия** текущего времени (если сегодня в пределах недели)
- Заголовки: день недели + число

**Отображение событий:**
- Пересекающиеся события в один день делят ширину колонки
- При 2 событиях → по 50% ширины
- При 3 событиях → по 33.3% ширины

**Совет**: Идеален для планирования рабочей недели.

### 4. Месяц (MonthView)

![Месяц]

**Особенности:**
- Сетка 6 строк × 7 дней (Пн – Вс)
- Число месяца в правом верхнем углу ячейки
- **Красный кружок** для сегодняшней даты
- Серый текст для дней другого месяца

**Отображение событий:**
- До 3 событий в ячейке
- Формат: "09:00 Название"
- При большем количестве: "+N ещё"
- Цветная левая полоска у каждого события

**Действия:**
- **Двойной клик** по ячейке → новое событие
- **Клик** по дате → (сигнал, пока не используется)

**Совет**: Лучший обзор на месяц вперёд.

### 5. Год (YearView)

![Год]

**Особенности:**
- 12 мини-календарей (3 строки × 4 колонки)
- Название месяца красным (если текущий)
- **Красный кружок** на сегодняшнем дне
- **Синие точки** под числами с событиями

**Навигация:**
- **Клик** по любому месяцу → переключение на MonthView с этим месяцем
- Кнопки **◀ / ▶** меняют год

**Совет**: Используйте для быстрого перехода к нужному месяцу.

## 🏷️ Категории

### Управление категориями

1. Нажмите кнопку **"Категории"** (рядом с "Событие")
2. Откроется диалог управления:

```
┌─────────────────────────────────┐
│     Управление категориями       │
├─────────────────────────────────┤
│ • Работа     [Синий]            │
│ • Личное     [Зелёный]          │
│ • Важное     [Красный]          │
│                                 │
│ Название: [________]            │
│ Цвет:     [▼ Синий]             │
│                                 │
│ [Добавить] [Изменить] [Удалить] │
│                     [Закрыть]   │
└─────────────────────────────────┘
```

### Действия с категориями

| Действие     | Как сделать                                        |
| ------------ | -------------------------------------------------- |
| **Добавить** | Введите название → выберите цвет → "Добавить"      |
| **Изменить** | Кликните по категории → измените поля → "Изменить" |
| **Удалить**  | Кликните по категории → "Удалить" → подтвердить    |

**Важно**: 
- Стандартные категории ("Работа", "Личное", "Важное") нельзя удалить
- При удалении категории все её события переходят в "Личное"

## 🧭 Навигация

### Кнопки навигации

| Кнопка      | Действие в разных режимах           |
| ----------- | ----------------------------------- |
| **Сегодня** | Возврат к текущей дате/месяцу/году  |
| **◀**       | Назад: день, неделя, месяц или год  |
| **▶**       | Вперёд: день, неделя, месяц или год |

### Горячие клавиши (планируются)

| Комбинация  | Действие                                                       |
| ----------- | -------------------------------------------------------------- |
| `Ctrl+N`    | Новое событие                                                  |
| `Ctrl+E`    | Редактировать событие                                          |
| `Delete`    | Удалить событие                                                |
| `Ctrl+1..5` | Переключить режим (1-Список, 2-День, 3-Неделя, 4-Месяц, 5-Год) |

## 💡 Советы и хитрости

### 1. Быстрое создание событий
- В режиме **Месяц**: дважды кликните по дате, укажите только время
- В режиме **День**: дважды кликните по времени на шкале

### 2. Планирование встреч
- Используйте категорию **"Работа"** для рабочих встреч
- Указывайте описание с местом встречи
- Создавайте событие на 30-60 минут с запасом

### 3. Отслеживание дедлайнов
- Создавайте событие **"Важное"** за день до дедлайна
- Ставьте время на 09:00 или 18:00
- Просматривайте в режиме **"Список"** каждое утро

### 4. Личные напоминания
- Категория **"Личное"** - зелёный цвет
- Добавляйте эмодзи в название: 🏋️ "Тренировка", 🛒 "Покупки"
- Используйте описание для списка дел

### 5. Обзор года
- В декабре смотрите **Годовой вид** для планирования отпусков
- Точки под числами показывают загруженные дни
- Клик по месяцу для деталей

### 6. Работа с overlapping событиями
- В режиме **День/Неделя** пересекающиеся события автоматически делят ширину
- Чем больше событий, тем уже каждый блок
- Кликните по любому блоку для редактирования

### 7. Поиск (будет в следующей версии)
- Пока используйте **Список** и прокрутку
- Или экспортируйте данные и ищите в текстовом редакторе

## ❓ Часто задаваемые вопросы

### Q1: Где хранятся мои данные?

**A:** 
- **Windows**: `C:\Users\ВашеИмя\AppData\Roaming\MyCalendar\calendar.db`
- **macOS**: `~/Library/Application Support/MyCalendar/calendar.db`
- **Linux**: `~/.local/share/MyCalendar/calendar.db`

### Q2: Как сделать резервную копию?

**A:** Скопируйте файл `calendar.db` в безопасное место. Для восстановления замените файл.

### Q3: Можно ли импортировать события из Google Calendar?

**A:** Пока нет, но функция планируется. Сейчас можно добавить события вручную.

### Q4: Почему событие не отображается в режиме "Месяц"?

**A:** В ячейке месяца показываются только события, начинающиеся в этот день. События, длящиеся несколько дней, отображаются только в первый день.

### Q5: Как изменить цвет категории после создания?

**A:** Нажмите **"Категории"** → выберите категорию → выберите новый цвет → **"Изменить"**.

### Q6: Можно ли удалить стандартные категории?

**A:** Нет, "Работа", "Личное" и "Важное" защищены от удаления. Вы можете их переименовать и изменить цвет.

### Q7: Приложение зависло. Что делать?

**A:** 
1. Закройте через Диспетчер задач (Ctrl+Shift+Esc)
2. Запустите заново
3. Если проблема повторяется, проверьте целостность БД (см. Q8)

### Q8: Как восстановить повреждённую базу данных?

**A:** 
1. Закройте приложение
2. Удалите `calendar.db`
3. Запустите приложение - создастся новая БД
4. Восстановите события из резервной копии

### Q9: Можно ли использовать приложение без мыши?

**A:** Частично. Основные функции доступны через Tab и Enter, но для полного удобства нужна мышь.

### Q10: Будет ли мобильная версия?

**A:** В планах - веб-версия и мобильное приложение на Flutter.
