# 📅 Календарь-Планировщик

Многофункциональное приложение-календарь с поддержкой категорий, разных режимов просмотра и удобным управлением событиями.

![Версия](https://img.shields.io/badge/version-1.0.0-blue)
![Python](https://img.shields.io/badge/python-3.8+-green)
![PyQt5](https://img.shields.io/badge/PyQt5-5.15+-orange)

## ✨ Возможности

- **5 режимов просмотра**: Список, День, Неделя, Месяц, Год
- **Управление категориями**: Создание, редактирование, удаление категорий с цветовой маркировкой
- **CRUD операции с событиями**: Создание, просмотр, редактирование, удаление
- **Визуальные эффекты**: Подсветка сегодняшнего дня, цветовые индикаторы, анимации
- **Хранение данных**: SQLite база данных с автоматическим сохранением
- **Навигация**: Кнопки "Сегодня", "Назад", "Вперёд"
- **Гибкая работа со временем**: Выбор даты и времени для каждого события

## 🎯 Использование

### Основные действия

- **Добавить событие**: Нажмите "Событие" → заполните форму → сохраните
- **Редактировать**: Дважды кликните по событию в любом представлении
- **Удалить**: Откройте событие → нажмите "Удалить"
- **Переключить вид**: Используйте сегментированный контрол в верхней панели
- **Навигация**: Кнопки "Сегодня", "◀", "▶" в навигационной панели
- **Управление категориями**: Нажмите "Категории" → добавьте/измените/удалите

### Режимы просмотра

| Режим      | Описание                                                         |
| ---------- | ---------------------------------------------------------------- |
| **Список** | Все события, сгруппированные по датам, с зачёркиванием прошедших |
| **День**   | Временная шкала 24 часа с событиями, линия текущего времени      |
| **Неделя** | Пн–Пт, временная шкала, события с overlapping-логикой            |
| **Месяц**  | Сетка 6x7, компактное отображение событий в ячейках              |
| **Год**    | 12 мини-календарей, точки событий, клик → месяц                  |

## 📁 Структура базы данных

### Таблица `events`
| Поле        | Тип                 | Описание               |
| ----------- | ------------------- | ---------------------- |
| id          | INTEGER PRIMARY KEY | Уникальный ID          |
| title       | TEXT NOT NULL       | Название события       |
| description | TEXT                | Описание               |
| start_dt    | TEXT (ISO)          | Дата и время начала    |
| end_dt      | TEXT (ISO)          | Дата и время окончания |
| category    | INTEGER             | FK на categories.id    |

### Таблица `categories`
| Поле  | Тип                 | Описание           |
| ----- | ------------------- | ------------------ |
| id    | INTEGER PRIMARY KEY | Уникальный ID      |
| name  | TEXT UNIQUE         | Название категории |
| color | TEXT                | HEX-код цвета      |
